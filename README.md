# kubernetes-intermedio-local

Laboratorio de Kubernetes Intermedio en entorno local utilizando Minikube y kubectl.

Se divide en dos partes:

- **ejemplo/**: Manifiestos de Kubernetes para desplegar una aplicación web simple con Nginx.
- **k8s-auth-demo/**: Manifiestos de Kubernetes para desplegar una aplicación de dos bases de datos para dos proyectos Spring Boot.
